From UI-SPEC Animation Contract (key entries):
- List items (filter change): staggered opacity + y offset, 150ms, 30ms stagger
- Floor plan card hover: scale 1->1.03 + shadow, spring, instant
- Timeline block hover: scale 1->1.02, CSS transition-transform duration-150
- Tab indicator: underline slides to active, motion layout span, 180ms spring
- Party size change: NumberTicker, 400ms
- Save button during submit: BorderBeam glow
- Presence badge editing: CSS animate-ping (already exists)

What NOT to animate:
- Virtualized timeline grid (DOM node recycling breaks AnimatePresence)
- Every list row on scroll
- Form inputs during typing
- Error messages (role="alert")
