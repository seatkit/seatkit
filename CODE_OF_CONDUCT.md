# SeatKit Code of Conduct

SeatKit is an open source, developer-oriented project. We are committed to a welcoming, harassment-free experience for everyone.

This project **adopts the [Contributor Covenant Code of Conduct, version 2.1](https://www.contributor-covenant.org/version/2/1/code_of_conduct/)**. The full text applies; key points and enforcement details are summarized below.

## Our Standards (summary)

Examples of behavior that contributes to a positive environment include:

- Being respectful and empathetic
- Giving and gracefully accepting constructive feedback
- Focusing on what is best for the community

Unacceptable behavior includes:

- Harassment, insults, or derogatory comments
- Trolling or sustained disruption
- Publishing others’ private information without consent

## Scope

This Code applies within all project spaces (GitHub issues/PRs, discussions, chat) and in public spaces when an individual is representing the project.

## Enforcement

Report incidents to **conduct@seatkit.dev** (replace if different). Maintainers will review and respond within **7 business days**.

### Enforcement Guidelines (summary)

Actions may include (progressively, as appropriate):

1. **Correction** — Private, written warning; request to stop.
2. **Warning** — Formal warning; behavior must change; no further contact with individuals affected.
3. **Temporary Ban** — Temporary removal from project spaces.
4. **Permanent Ban** — Permanent removal from project spaces.

Reports will be handled confidentially. Retaliation against reporters is not tolerated.
